Webscrapping.

Projet de Webscrapping, développement d'un script en Python permettant d'extraire des données depuis un site web - Juillet 2022
Python BeautifulSoup
❮ ❯

Contexte du projet

J'ai développé ce script de webscrapping suite à la demande d'une connaissance qui souhaitait établir une liste de prospects. Il permet d'extraire toutes les informations disponibles (nom, adresse, numéro, coordonnées de l'agence, information des salariés...) liées aux agences immobilières présentes sur le site web "nestenn.com".

N'ayant jamais fait de webscrapping avant, j'ai d'abord du me renseigner et me former. J'ai utilisé la bibliothèque Python BeautifulSoup4 qui permet de faire une analyse syntaxique de documents HTML et XML. Le script permet d'itérer parmi toutes les pages de toutes les agences présentes et de "creuser" dans chacune d'entre elles pour y extraire les informations et les ajouter dans un fichier csv, ce qui permet ainsi de les consulter et de les manipuler plus facilement sur Excel.
